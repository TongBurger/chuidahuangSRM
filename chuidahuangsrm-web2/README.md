# CDH SRM Demo 前端页面

## 说明

由于无法直接获取 `https://melconnele4.github.io/cdh-srm-demo/` 的源代码（可能原因：仓库已设为私有、已删除或未通过标准GitHub方式部署），我基于获取到的网站内容结构创建了一个静态HTML版本。

## 文件说明

- `index.html` - 主页面文件，包含完整的SRM系统界面结构

## 功能模块

1. **工作台** - 供应链运营概况，包含KPI指标、预警提醒和近期订单
2. **订单协同** - U8同步的采购订单与委外订单管理
3. **委外管理** - 虚拟仓库存管理，BOM驱动发料与物料核销
4. **送货管理 (ASN)** - 发货通知单管理，标签打印与扫码收货
5. **模具管理** - 模具全生命周期管理，一模一码追溯
6. **供应商管理** - 供应商全生命周期管理，准入审核、资质预警
7. **寻源管理** - 询比价、在线招标与成本构成分析
8. **财务对账** - 自动对账、三单匹配、发票管理
9. **合同管理** - 合同模板管理、在线审批与电子签章

## 技术特点

- 纯静态HTML + CSS + JavaScript
- 无需构建工具，直接在浏览器中打开即可使用
- 响应式设计，支持移动端访问
- 现代化的UI设计，使用CSS变量和Flexbox/Grid布局

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：`python -m http.server 8000` 然后访问 http://localhost:8000

## 注意事项

- 这是一个基于原网站内容结构的重建版本，不是原始源代码
- 数据展示为静态示例，不包含实际的后端API对接
- 如需获取原始源代码，建议联系 GitHub 用户 `melconnele4` 获取仓库访问权限
